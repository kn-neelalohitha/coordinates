= getAngle()

=== Description

Reads the angular coordinate of the point in radians.

=== Syntax

`getAngle()`

=== Parameters

Nothing

=== Returns

`phi`: the angular coordinate of the point in radians (`float`)

=== Example

[source,cpp]
----

# include <Coordinates.h>
Coordinates point = Coordinates();
void setup() {
  // put your setup code here, to run once
  Serial.begin(9600);
  float phi;
  phi = point.getAngle();
  Serial.print("Ï†: ");
  Serial.println(phi);
  Serial.println(" rads");
}

void loop() {}

----
